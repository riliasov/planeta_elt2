Этот воркфлоу устанавливает зависимости, запускает обновление данных (ELT) и поднимает дашборд.

1. Установка зависимостей
// turbo
pip install -r requirements.txt

2. Запуск ELT пайплайна
# Флаги:
# --full-refresh : полная перезагрузка данных
# --deploy-schema : пересоздание таблиц стейджинга
python src/main.py

3. Запуск дашборда
streamlit run dashboard.py

4. Завершение
Если все шаги прошли успешно и без критических ошибок, НЕ СОЗДАВАЙ artifacts/walkthrough.md.
